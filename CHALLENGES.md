# Retos adicionales

Este archivo incluye retos opcionales para los lectores que quieran profundizar más allá de los ejercicios incluidos en cada capítulo.

Puedes intentar resolverlos en:

- Kotlin
- Swift
- Dart

---

## Reto 1 — Calculadora básica

Crea una calculadora que permita realizar las siguientes operaciones:

- suma
- resta
- multiplicación
- división

Bonus:

- manejar división por cero
- permitir números negativos

---

## Reto 2 — Sistema de notas de estudiantes

Crea un pequeño programa que permita:

- almacenar estudiantes
- registrar notas
- calcular el promedio
- determinar si el estudiante aprueba o reprueba

Bonus:

- ordenar estudiantes por nota
- encontrar la nota más alta

---

## Reto 3 — Sistema de inventario

Crea un pequeño sistema de inventario con productos.

Propiedades del producto:

- nombre
- precio
- cantidad

Funciones:

- agregar producto
- eliminar producto
- calcular valor total del inventario

---

## Reto 4 — Simulación de petición async

Simula una petición de red que:

- obtenga una lista de usuarios
- espere unos segundos
- imprima los resultados

Bonus:

- ejecutar múltiples tareas async
- combinar resultados

---

## Reto final

Crea un pequeño proyecto combinando varios conceptos del libro.

Ideas posibles:

- gestor de tareas
- catálogo de productos
- aplicación de notas
- gestor de contactos

Utiliza conceptos como:

- clases
- colecciones
- funciones
- manejo de errores
- asincronía

---

Programming mastery comes from **building real things**.
